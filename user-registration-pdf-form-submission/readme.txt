PDF form submission addon for user registration plugin.

== Description ==

Export user's data in PDF. PDF file Email attachment

Get [free support](https://wpuserregistration.com/support/)


### Features And Options:
* Export user's data in PDF form
* PDF file email attachment for admin
* PDF file email attachment for user
* WooCommerce fields support
* Incredible Support
* Well Documented
* Translation ready
