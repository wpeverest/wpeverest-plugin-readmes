Profile Connect addon for User Registration

== Description ==

User Registration Profile Connect allows you to connect already registered users ( by other than user registration form ) to the user registration form so that the user/admin can later update the form fields. So, this add-on helps users to build up the profile with user registration fields even if they are registered via other means.
Get [free support](https://wpuserregistration.com/support/)


### Features And Options:
* Connect form to previously registered users.
* Build profile with user registration form fields for already registered users.
* Bulk Users connection
* Incredible Support
* Well Documented
* Translation ready
