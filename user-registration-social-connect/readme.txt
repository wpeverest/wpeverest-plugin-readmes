== Description ==

Social connect addon for user registration plugin.

Get [free support](https://wpuserregistration.com/support/)


### Features And Options:
* Login with facebook,twitter,google and linkedin
* Multiple social template
* Social registration
* Incredible Support
* Well Documented
* Translation ready
